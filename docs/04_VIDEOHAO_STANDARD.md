# 04 视频号标准

## 内容定位

视频号内容应服务财经解释和行业判断，重点是把公众号文章转成短视频可理解的观点结构。

## 时长

- 90s：适合单观点、强钩子、快速解释。
- 180s：适合多层分析、事实补充和风险提示。

## 表达要求

- 开场 3 秒内给出明确问题或判断。
- 观点必须来自文章正文，不编造融资金额、投资方、客户或收入。
- 信息图优先使用自制图表，不依赖新闻原图。
- 结尾应收束到行业信号或编辑判断。

## 视觉要求

- 竖屏优先。
- 画面信息密度适中，避免长段文字堆叠。
- 不使用真实公司 Logo、真实新闻图、真实视频素材作为 Batch 01 假数据。
- 所有素材必须经过版权策略检查。
