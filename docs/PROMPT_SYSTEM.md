# Prompt 系统边界

Batch 00 不实现完整 AI 生成链路，也不固化最终提示词。本文档只定义后续 prompt 模块应覆盖的边界，避免第一版 schema 和后续生成流程脱节。

## 后续模块

- 文章理解：提取事实、事件、公司、行业标签和关键数字。
- 观点生成：把文章内容转成适合视频号表达的核心观点。
- 脚本生成：分别生成 90s 与 180s 脚本。
- 分镜生成：把脚本拆成可执行镜头、画面和素材需求。
- 图表建议：识别适合图表化表达的数据和对比关系。
- AI 图像 Prompt：生成不依赖未授权素材的原创画面提示词。
- AI 视频 Prompt：生成适合竖屏短视频的动态画面提示词。
- 素材搜索线索：输出搜索关键词、平台建议和授权要求。
- 版权风险检查：标出可能涉及新闻图、影视片段、音乐、字体、商标或肖像的风险。
- 发布文案：生成封面文案与视频号发布文案。

## Prompt 输入边界

后续 prompt 应只依赖经过 `ArticleInputSchema` 校验的用户输入。系统不应自动抓取公众号全文，也不应把未校验的外部网页内容混入生成链路。

## Prompt 输出边界

后续 prompt 的结构化输出必须能通过 `VideoProductionPackageSchema` 校验。无法确定的事实、素材授权或版权状态应在对应字段中提示人工确认，而不是假定可用。

## 内容原则

- 财经判断必须基于文章正文，不编造融资金额、投资方、收入规模或客户名称。
- 脚本表达可以口语化，但不能夸大事实或制造未经证实的结论。
- 图像和视频 Prompt 应避免真实新闻图复刻、品牌标识复刻和影视风格复刻。
- 版权风险必须显式列出，不应隐藏在素材建议里。
