# 独角兽早知道 Video Agent MVP PRD

## 目标

第一版产品将人工提供的公众号财经文章转化为微信视频号生产包，帮助编辑快速完成短视频选题拆解、脚本撰写、分镜规划、素材准备和发布文案准备。

Batch 00 的目标是建立项目骨架和文档系统，不接入 AI 模型，不生成完整成片。

## 用户与场景

目标用户是《独角兽早知道》的内容编辑、视频编导和运营人员。典型场景是编辑拿到一篇公众号文章后，需要快速判断视频号表达角度，并交付给剪辑或素材制作人员一套结构化生产包。

## 第一版输入

- `title`：公众号文章标题
- `body`：公众号文章正文，由用户人工粘贴
- `url`：文章链接
- `publishedAt`：发布日期，格式为 `YYYY-MM-DD`
- `source`：文章来源
- `industryTags`：行业标签
- `targetDuration`：目标时长，只支持 `90` 或 `180`

## 第一版输出

- 核心摘要
- 核心观点
- 90s 视频号脚本
- 180s 视频号脚本
- 分镜表
- 图表建议
- AI 图像 Prompt
- AI 视频 Prompt
- 素材搜索线索
- 版权风险表
- 封面文案
- 发布文案
- Markdown / CSV / JSON 导出包

## 用户流程

1. 编辑输入文章标题、正文、链接、发布日期、来源、行业标签和目标时长。
2. 系统校验输入字段是否符合 schema。
3. 后续版本调用内容生成流程，输出结构化视频号生产包。
4. 编辑人工审核观点、事实、版权风险和发布文案。
5. 编辑导出 Markdown、CSV 或 JSON，交给剪辑、设计或运营使用。

## 非目标

- 不自动抓取公众号全文
- 不自动下载网络素材
- 不自动发布视频号
- 不自动生成完整成片
- 不使用未确认版权的新闻图、视频片段、影视片段、音乐和字体

## Batch 00 验收标准

- `pnpm install` 可安装依赖。
- `pnpm dev` 可启动状态页。
- `pnpm test` 校验 schema 和样例数据。
- `pnpm lint` 和 `pnpm build` 可作为基础质量门禁。
- 文档说明 MVP 范围、输出结构、版权策略和 prompt 系统边界。
